# seleniumTest
## How To Execute

In Command prompt  
  1. install chromedriver to path ```/usr/bin/chromedriver/chromedriver_linux64/```
  2. use ```mvn clean test``` to execute the test
  
In IDE 
  1. install chromedriver to path ```/usr/bin/chromedriver/chromedriver_linux64/```
  2. Run ```AllTests.java``` as Junit test
